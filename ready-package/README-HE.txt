# תיקייה זו מוכנה לשימוש עם תוכנות יצירת התקנה

## תוכן התיקייה:
- main.js - קובץ האלקטרון הראשי
- package.json - הגדרות האפליקציה  
- dist/ - האפליקציה הבנויה (יתווסף אוטומטית)
- assets/ - תיקיית אייקונים

## איך להשתמש:
1. העתק את תיקייה dist מהגיטהאב Actions לכאן
2. גרור את כל התיקייה לתוכנת Packages 
3. בנה את המתקין

הכל מוכן!
